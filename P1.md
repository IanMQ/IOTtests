## 🔍 Instrucciones de búsqueda (qué buscar y dónde)

| Si buscas... | Lo encuentras en... | Lo escribes como... |
|--------------|---------------------|----------------------|
| **Voltaje** | Datasheets → "Supply Voltage", "VDD", "Operating Voltage" | `Se requiere 5V ±0.2V` |
| **Corriente normal** | Datasheets → "Supply current", "Operating current" | `Consumo típico de 80 mA` |
| **Pico de corriente** | Datasheets → "Inrush current", "Peak current" | `Debe soportar picos de 600 mA` |
| **Tiempo de conversión** | Datasheets → "Conversion time", "Response time", "tCONV" | `Tiempo máximo de 750 ms` |
| **Latencia** | Caso → busca: "tiempo real", "periódicamente", "respuesta automática" | `Se requiere latencia máxima de X ms` |
| **Justificación** | Caso → busca: "riesgo", "salud", "alerta", "ocupantes" | `para alertar oportunamente sobre condiciones de riesgo` |

## ✅ Ejemplo llenado (caso AirWatch)

| System Requirement | Especificación de requisitos |
|-------------------|-------------------------------|
| **Capacidades de suministro de energía** | Según el datasheet del sensor HPMA115S0, se requiere **5V ±0.2V**. El caso no especifica batería, por lo que se asume fuente externa continua. Debe soportar picos de **600 mA** (inrush current del HPMA115S0 según su datasheet). |
| **Restricciones de time-delay** | Según el datasheet del sensor DS18B20, el tiempo de conversión a 12 bits es de **750 ms** (máximo). Se requiere que el ciclo completo de lectura, envío al Edge Gateway y activación del LED no supere este valor, para garantizar que las alertas por condiciones de riesgo (CO₂, PM2.5) sean oportunas y protejan la salud de los ocupantes según lo indicado en el caso. |

## ⚡ Reglas rápidas (cheat sheet)

| Regla | Ejemplo |
|-------|---------|
| **Voltaje** siempre con `±` y unidad | `5V ±0.2V`, `3.3V ±5%` |
| **Corriente** en `mA` (no A) | `80 mA`, `600 mA` |
| **Tiempos** en `ms` o `s` | `750 ms`, `< 6 s` |
| **Usa "según datasheet"** para respaldar | `según datasheet del DS18B20` |
| **Usa "según el caso"** para contexto | `según el caso, para proteger la salud de los ocupantes` |
| **No inventes números** | Si no está en datasheet ni caso, no lo pongas |
