## ⚡ Reglas rápidas (cheat sheet) para Pregunta 3

| Regla | Ejemplo |
|-------|---------|
| **Usuarios:** copiar textual del caso | `facility managers, responsables de seguridad ocupacional, auditores` |
| **Servicios:** buscar verbos de acción en el caso | `monitorear, recibir alertas, consultar historial, acceder a recomendaciones` |
| **Necesidades de información:** conectar servicio con datos | `alerta necesita umbrales, monitoreo necesita datos de sensores` |
| **Alinear con estándares mencionados** | `ASHRAE 62.1, OMS, WELL, LEED` |
| **No inventar usuarios** | Si el caso no lo menciona, no lo pongas |
| **Servicios pueden repetirse entre usuarios** | `todos usan monitoreo, no todos usan historial` |
